Reaction Roles

Let members self-assign roles by reacting to a message with an emoji. Great for opt-in channels and interest groups.

🎭
Configuration
Set up the reaction roles message and channel
Enable Reaction Roles
Members can self-assign roles via emoji reactions
Leave blank — StoatMod will post a new message for you
➕
Emoji → Role Mappings
Add one row per role. Click the emoji to change it.
🔒
Requires Bot Admin Setup
Reaction roles require StoatMod to have Manage Roles permission and to be in your server. Run /reactionroles apply to publish the role message.
